# Seven fish 前端UI框架
## <qgy-button>组件
+ iconClass：自定义图标名称，默认为空
+ type:string类型，接受参数有default,primary,success，默认为default
+ iconDirection:图标位置,接受参数有left,right,默认为left
+ loading:是否显示loading，接受一个boolean，默认为false
## <qgy-link>组件
+ type:string类型，接受参数有default,primary,success，默认为default
+ link:string类型，接受一个链接
